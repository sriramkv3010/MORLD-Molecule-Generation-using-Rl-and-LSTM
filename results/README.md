# Results

This directory contains all output files from training and evaluation.

## Subdirectories:

### figures/
Contains all visualization plots (27 PNG files):
- Training metrics
- Evaluation metrics  
- Molecular property distributions
- Generated molecule visualizations

### logs/
Contains metrics and log files:
- `classification_report.txt` - Classification metrics
- `molecular_weights.csv` - Molecular weight data
- `reward_metrics_log.csv` - RL reward logs
- `test_metrics.pkl` - Test metrics

## Files:
- `generated_smiles_CN.smi` - Generated SMILES strings

# Source Code

This directory contains all Python source code files for the molecular generation pipeline.

## Files:
- `preprocess.py` - Data preprocessing
- `tokenize_smiles.py` - SMILES tokenization
- `split_data_tokens.py` - Data splitting
- `train_model.py` - LSTM model training
- `RLfinetune.py` - Reinforcement learning fine-tuning
- `testRLLSTM.py` - Testing and evaluation
- `comparision.py` - Model comparison utilities
- `molwtdist.py` - Molecular weight distribution analysis
- `plot.py` - Plotting utilities
- `test.py` - Testing utilities
